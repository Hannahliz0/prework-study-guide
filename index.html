HTML

  • The head element contains information about the webpage.
  • The body elements represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git stats: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git ad -A : we sure the command to commit all files in the working branch
  • git commit -m "your comment"
  • git pull origin main
  • git push origin "branch"

JavaScript

  • We can use the console.log method to output a message to the web console by adding an argument
  • The strict equality operator (===) checks to see if two values are equal, and returns a Boolean result true if the values are equal and false if the values are not equal
  • JavaScript code is executed from top to bottom. However, using conditional statements such as if statements allows us to interrupt control flow.
  • To declare a function, we use the function keyword and then give our function a unique name followed by parentheses. Then we add curly brackets {} that contain the code for the task we want to perform.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
